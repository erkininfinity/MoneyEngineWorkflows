# Hot Lead Detector Workflow

This workflow processes incoming leads from a webhook (e.g. connected to your landing page form, Typeform, or active email inbox hook), scores purchase intent, extracts estimated budget or urgency indicators, and sends a prioritized notification to your Telegram chat with a ready-to-send draft reply.

## Required integrations

- **n8n** (v1.0+)
- **OpenAI API**
- **Telegram Bot API**

## Setup

1. Import `workflow.json` into n8n.
2. Link your credentials.
3. Configure the `TELEGRAM_CHAT_ID` environment variable.
4. Set the webhook path or use the test webhook URL generated by n8n.
5. Point your form tool or lead capture pipeline to send POST JSON requests to the n8n webhook URL.

## Safety

This workflow is categorized under **Low Risk**. It analyzes incoming inquiries and drafts suggested replies. It does not send any reply messages automatically.
